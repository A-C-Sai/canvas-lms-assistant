# Agentic RAG Chatbot Development

A comprehensive project showcasing the development process of an intelligent chatbot assistant (ARTIM) for Canvas LMS using advanced RAG (Retrieval Augmented Generation) techniques and agentic behaviors.

## Project Structure

- `00-indexing/`: Data indexing and knowledge base setup with ChromaDB
- `01-synthetic-data-generation/`: Generation and validation of synthetic training data using deepeval
- `02-retrieval-evaluation/`: Evaluation and improvement of retrieval mechanisms
- `03-mvp/`: Initial MVP implementation with boto3/AWS Bedrock and terminal based UI and testing
- `04-generation-evaluation/`: Assessment and optimization of response generation
- `05-final-product/`: Near "Production-ready" implementation using LangGraph and Streamlit

Each subdirectory contains detailed documentation about its specific components and implementation details.
