# RAG Techniques Experiments

This directory contains a series of Jupyter notebooks exploring various Retrieval Augmented Generation (RAG) techniques and optimizations. Each notebook demonstrates different approaches to improve the accuracy and reliability of RAG systems.

## Overview

### Basic Implementations
1. **[00-Basic-RAG.ipynb](00-Basic-RAG.ipynb)**
   - Foundation of RAG implementation
   - Basic vector store setup
   - Simple similarity search retrieval
   - Standard prompt templates

2. **[01-Simple-CSV-RAG.ipynb](01-Simple-CSV-RAG.ipynb)**
   - Working with structured CSV data
   - Retrieval with structured content

### Advanced Techniques

3. **[02-Reliable-RAG.ipynb](02-Reliable-RAG.ipynb)**
   - Multi-step retrieval process
   - Self-verification mechanisms
   - Confidence scoring
   - Response validation

4. **[03-Optimizing-Chunk-Sizes.ipynb](03-Optimizing-Chunk-Sizes.ipynb)**
   - Empirical chunk size optimization
   - Impact on retrieval quality
   - Trade-offs analysis

5. **[04-Proposition-Chunking.ipynb](04-Proposition-Chunking.ipynb)**
   - Semantic-based text splitting
   - Proposition extraction

### Query Enhancement

6. **[05-Query-Transformations.ipynb](05-Query-Transformations.ipynb)**
   - Query expansion techniques
   - Query decomposition
   - Sub-query generation

7. **[06-Hypothetical-Document-Embedding-HyDE.ipynb](06-Hypothetical-Document-Embedding-HyDE.ipynb)**
   - HyDE implementation
   - Synthetic document generation
   - Enhanced semantic matching
   - Performance comparison with standard RAG

8. **[07-Hypothetical-Prompt-Embeddings-HyPE.ipynb](07-Hypothetical-Prompt-Embeddings-HyPE.ipynb)**
   - HyPE methodology
   - Prompt-based embeddings
